# Funcionalidades

## Galería dinámica
Las imágenes se generan dinámicamente desde un array de datos en JavaScript, simulando una API.

## Modal / Lightbox
- Apertura al hacer click en una imagen
- Bloqueo de scroll
- Navegación con botones y teclado
- Contador de imágenes

## Likes y dislikes
- Estado gestionado en memoria
- Exclusividad like / dislike
- Animaciones visuales

## Dark / Light mode
- Modo oscuro por defecto
- Cambio dinámico mediante variables CSS
